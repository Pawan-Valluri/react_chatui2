# Paragraph Styles Configuration
This file documents the available paragraph styles and their recommended usage within the document. You must only use these styles when applying formatting.

- **Normal**: Base font. *Recommendation*: Use for all standard body text and conversational output.
- **Heading 1**: Aptos 20pt, Dark Blue. *Recommendation*: Use ONLY for the main sections of a generated report.
- **Heading 2**: Aptos 16pt, Dark Blue. *Recommendation*: Use for major subsections within a Heading 1 section.
- **Heading 3**: 14pt, Dark Blue. *Recommendation*: Use for minor subsections.
- **Heading 4**: Italic, Dark Blue. *Recommendation*: Use for deep nesting or specific callout headers.
- **Title**: Aptos 28pt. *Recommendation*: Use exactly once at the very beginning of a formal document.
- **Subtitle**: 14pt, Gray. *Recommendation*: Use directly below the Title to provide context.
- **Quote**: Italic, Dark Gray, Center aligned. *Recommendation*: Use for blockquotes, important callouts, or verbatim excerpts.
- **List Paragraph**: Left indent 36pt. *Recommendation*: Use for bulleted items.
- **Numbered List**: Left indent 36pt. *Recommendation*: Use specifically for numbered items or procedural steps.
- **Subheading**: 10.5pt, Bold, Navy Blue. *Recommendation*: Use for small inline headers or metadata keys (e.g., "Author:", "Date:").
- **Section Heading**: Aptos. *Recommendation*: Use for separating content blocks within a section without needing a numbered hierarchy.
- **Review Title**: 12pt, Bold. *Recommendation*: Use for review sections, summaries, or emphasis blocks.
