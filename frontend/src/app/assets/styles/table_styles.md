# Table Styles Configuration
This file documents the available table styles and their recommended usage within the document. You must only use these styles when creating tables.

- **Table Grid**: Standard table with full borders. *Recommendation*: Preferred for generic data presentation and standard reporting.
- **Heading Table**: Table with emphasized header row formatting. *Recommendation*: Use when the first row contains critical category headers.
- **Heading Table 2**: Alternative table with header emphasis. *Recommendation*: Use as a stylistic alternative to Heading Table.
- **Title Table**: Table used for layout near the document title or metadata. *Recommendation*: Use for document metadata (Author, Date, Version) at the top of a report.
- **Totals 2** / **Totals Table 2**: Tables with emphasized bottom rows. *Recommendation*: Use for financial data, invoices, or any data where the final row represents a sum or conclusion.
- **Distribution List Table**: *Recommendation*: Use specifically for contact lists, action items assigned to people, or distribution tracking.
- **Normal Table**: Default unformatted table. *Recommendation*: Avoid unless a completely invisible layout grid is explicitly requested.
